# Changelog

## 0.1.0-alpha.0 - 2026-06-17

- Scaffolded the Director CLI and SDK for local-first live-app presentation production.
- Added storyboard validation, Playwright and OBS camera rig providers, ffmpeg final-cut rendering, PPTX pitch-deck rendering, manifests, and dailies.
- Added local MCP tools for agent-driven scouting, drafting, checking, shooting, rendering, and dailies retrieval.
- Added an A2A-shaped Director Agent adapter with task lifecycle states, approval gates, and artifact mapping.
- Added contract coverage for storyboard parsing, media arguments, Director drafting, MCP handlers, and A2A task behavior.
- Added public alpha release metadata, governance files, CI, packaging allowlist, release docs, and security sweep guidance.
